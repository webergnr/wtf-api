# WTF Api

REST API for the World Texting Foundation, also known as WTF.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install everything.

```bash
yarn install
```

## Creating / Seeding the database

```bash
yarn db:generate
yarn db:seed
```

## Testing

```bash
yarn test
```

## Running

```bash
yarn dev
```
